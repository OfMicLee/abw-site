公司简介

    南京全买酒进出口贸易有限公司是由旅意华侨陈岚女士经营的一家以葡萄酒为主营业务的品牌酒业公司。其在意大利经营的“憨憨”品牌凭借良好的信誉及独到的经营方式名振欧洲华人界。现利用其资源优势倾力打造的“lucia”品牌，主要致力于意大利各大著名酿酒家族的优质葡萄酒及其文化的推广。所售酒品均采购自实力情。雄厚、品质卓越的葡萄酒公司并直接原装原瓶进口，并通过在业内令人瞩目的表现更是国内同类产品中最具档次、品质最好，价格最实惠的意大利产品。
